== Description ==

* Author: [Fahad Mahmood](https://www.androidbubbles.com/contact)

* Project URI: <http://androidbubble.com/blog/wordpress/plugins/wc-order-split>
 
 
Create custom group labels and split WooCommerce orders.

After activation there will  be a Split icon in wp-admin > WooCommerce > orders list page within the order actions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There will now be a Split icon in the to Woocommerce  order overview page within the order actions.


== Frequently Asked Questions ==
= How to split orders with WC Order Split? =
After activation, a green icon will appear in orders list under actions. If actions do not appear, turn ON actions from screen options (right top). Click green icon, choose any group label and then select items to split in a new order. Proceed to split.

= How to define group labels? =
Go to plugin settings page. Enter label title, select text and background color then press add group.

= If I do not select any group label, what will happen? =
In case you do not select any group label, still it will automatically pick the first label as default.
